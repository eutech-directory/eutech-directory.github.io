What is Legis-Link?

A free compliance tool you open in your phone browser — no app to install.

Ask a compliance question in plain English
Type your question the way you'd ask a colleague — "what wire size for 45A over 30 metres?" — and get an instant answer with the exact code reference and clause. Every answer tells you which standard it came from and links to the full document.
Built for the job site, not the office
Works on any phone, any browser. No app store, no account required to use the chat. Takes 5 seconds to open and get an answer. Results are formatted for a small screen.
Every check is logged automatically
Every compliance check creates a tamper-proof audit record. Foremen can seal records and generate printable compliance certificates. PMs can view the full audit log from the dashboard.

Compliance certificates

Every sealed record generates a printable certificate you can hand to an inspector or PM.

How to get a certificate
Run a compliance check → note the Event ID in the result → go to /certificate/YOUR-EVENT-ID → print or save as PDF. The certificate shows the result, code citation, verification URL, and a cryptographic audit ID that proves it hasn't been altered.

Tips for best results

The tool understands plain English — these habits get you faster, better answers.

✓

Include the numbers

"Wire size for 45A, 30m, 240V" gives a precise result. "What size wire do I need?" gives a general answer. The more specific you are, the more accurate the calculation.

✓

Set your location first

Always pick your jurisdiction before asking questions. The same load in NSW and Texas gets different conductor sizes from different standards — the location is what determines which code applies.

✓

Use the quick-action buttons

After setting context, tap the chips at the bottom of the screen — Wire sizing, Load calc, Battery clearance, etc. They pre-fill a useful question template for your trade.

!

Always verify before ordering materials

Legis-Link results are preliminary checks based on published standard tables. For final installation decisions, confirm against the full published standard or consult a licensed engineer.
